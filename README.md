# Student Performance Intelligence Dashboard

This project analyzes student performance using Google Sheets.

Features:
- Final student score combining marks and attendance
- Relative grading system
- Identification of at-risk students
- Mentor-wise performance analysis
- Stream-wise performance comparison
- Admission channel vs student performance correlation

Tools Used:
Google Sheets, Pivot Tables, Data Visualization
